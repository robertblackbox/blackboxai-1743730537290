# Uber Clone

## Project Overview
Uber Clone is a web application that mimics the Uber platform's user interface. Built with HTML and Tailwind CSS, this project aims to provide users an intuitive and responsive experience for requesting rides, exploring services, and accessing business solutions. 

## Installation
To run this project locally, you need to have a web server. You can simply open the `index.html` file in your browser, or use a local server like Live Server in Visual Studio Code. Here’s how to set it up:

1. **Clone the repository** (if applicable):
   ```bash
   git clone <repository-url>
   cd uber-clone
   ```
2. **Open the project in your preferred web server**, or just open `index.html` in a web browser.

## Usage
Once the project is running, simply open the `index.html` in your browser. You’ll see the main interface where you can interact with features such as:

- Inputting pickup and dropoff locations.
- Viewing various services offered (Ride, Uber Eats, Freight).
- Navigating through different sections to learn more about Uber for Business and their offerings.

## Features
- Responsive design built with Tailwind CSS
- Intuitive layout mimicking the Uber application
- Interactive elements for user engagement
- Promo banner for promotional codes
- Easy navigation across services and company information

## Dependencies
This project currently includes the following dependencies as referenced in the HTML file:
- **Tailwind CSS**: A utility first CSS framework for designing fast and easy responsive layouts.
- **Font Awesome**: Provides vector icons and social logos for easy integration.
- **Google Fonts**: For enhanced typography using the Inter font family.

## Project Structure
The project contains the following key structures:
```
/
├── index.html                # Main HTML file for the application
```

### Key Sections in index.html:
- **Head Section**: Contains meta tags, title, linked stylesheets, and JavaScript.
- **Body Section**: Contains:
  - **Promo Banner**: A promotional message for users on discounts.
  - **Header**: Contains navigation links and logo.
  - **Main Content**:
    - **Hero Section**: Introduction with a background image and input fields.
    - **Services Section**: Displays available services.
    - **Business Section**: Information about Uber for Business.
  - **Footer**: Includes links to corporate information and social media.
